# 手机银行系统

​		一个基于 Python 开发的命令行手机银行系统，支持用户注册、登录、存取款、转账、查询等核心 banking 功能。

## 功能特性

| 功能模块 |                   描述                   |
| :------: | :--------------------------------------: |
| 用户认证 |      注册、登录、退出登录、注销账户      |
| 资金管理 |        存款、取款、转账、余额查询        |
| 交易记录 |             查看账户明细流水             |
| 安全设置 | 修改密码、密码错误锁定机制、密码加密存储 |

## 系统要求

- python 3.6+
- 无需第三方依赖（仅使用标准库）

## 快速开始

### 1.克隆项目

```
git clone https://github.com/ren-xiaolu/mobile_banking_system
cd mobile-banking-system
```

### 2.运行系统

```
python run.py
```

## 功能说明

### 用户注册

- 输入手机号（自动格式校验）

- 输入用户名、密码、确认密码

- 检测手机号是否已注册

- **安全机制**：密码加密存储

- 新用户数据保存至 `user_data/` 目录，以手机号命名

### 用户登录

- 输入手机号和密码进行验证
- **安全机制**：密码连续输错 3 次自动退出系统
- 未注册用户自动引导至注册流程

### 存款

- 输入存款金额（自动校验金额合法性）
- 实时更新余额并记录交易流水

### 取款

- 输入取款金额
- 校验余额是否充足
- 更新余额并记录流水

### 转账

- 输入收款人手机号和用户名验证
- 输入转账金额和密码
- **安全机制**：密码连续输错 3 次返回主页
- 实时扣款并记录双方流水

### 查询功能

- 查看余额：显示当前账户余额
- 查看明细：展示完整交易流水记录

### 账户管理

- 修改密码：需验证原密码，新密码不可与原密码相同
- 退出登录：清除登录状态，返回主菜单
- 注销用户：永久删除账户数据文件

## 数据存储

每个用户数据文件包含：

- 用户名
- 手机号
- 密码（加密存储）
- 账户余额
- 交易明细记录

```
user_data/
├── 13800138000.json    # 用户数据文件（手机号命名）
├── 13900139000.json
└── ...
```

## 安全特性

- 手机号格式合法性校验
- 密码强度基础校验
- 交易密码二次确认
- 连续错误密码锁定机制（3次）
- 余额充足性校验
- 收款人信息双重验证
- 密码加密存储
