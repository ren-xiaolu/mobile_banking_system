# 手机银行系统

功能如下：

## 0.注册

```
-> 输入手机号
-> 手机号格式判断
-> 输入用户名、密码、确认密码
-> 两次密码一致性判断
-> 判断手机号是否注册过
-> 将未注册的用户数据保存到user_data中，以手机号命名；
手机号已注册则登录
```

## 1.登录

```
-> 输入手机号、密码
-> 用户已存在则判断密码是否一致，密码输错三次则退出系统；
用户不存在则注册
```

## 2.退出系统

## 3.存钱

```
-> 输入存款金额
-> 金额合法性判断
-> 更改余额、添加流水、保存更改数据
```

## 4.取钱

```
-> 输入取款金额
-> 金额合法性判断
-> 判断余额是否足够
-> 更改余额、添加流水、保存更改数据
```

## 5.转账

```
-> 输入收款人手机号
-> 手机号格式判断
-> 输入收款人用户名
-> 判断收款人信息是否正确
-> 输入汇款金额
-> 金额合法性判断
-> 输入密码
-> 判断余额是否足够
判断密码是否正确;连续3次错误返回主页;
-> 更改余额、添加流水、保存更改数据
```

## 6.查看余额

```
-> 根据登录的手机号获取用户名、余额，并返回
```

## 7.查看明细

```
-> 根据登录的手机号获取明细，并返回
```

## 8.修改密码

```
-> 输入原密码、新密码
-> 两次密码不能一致、原密码是否正确
-> 更改密码，保存更改数据
```

## 9.退出登录

```
-> 更改登录状态标识
```

## 10.注销用户

```
-> 根据登录状态标识找到用户数据文件
-> 删除用户数据文件
-> 更改登录状态标识
```
