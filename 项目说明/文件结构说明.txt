- api 处理业务逻辑
    - bank.py 银行的业务逻辑
        存钱：save_money_info(phone_number, money)
        取钱：get_money_info(phone_number, money)
        收款人信息确认：recipient_info(recipient_phone, recipient_name)
        转账：transfer_account_info(recipient_phone, phone_number, password, money)
        查看明细：check_bill_info(phone_number)
    - user.py 用户的业务逻辑
        注册：register_info(phone_number, name, password)
        登录：login_info(phone_number, password)
        查看余额：check_money_info(phone_number)
        更改密码：change_password_info(phone_number, old_password, new_password)
        注销用户：delete_user_info(phone_number)
- config 配置信息(数据库位置，数据库账号密码)
    - setting.py
- core 核心文件，代码基本功能实现
    -src.py
        注册：register()
        登录：login()
        退出系统：sys.exit()
        存钱：save_money()
        取钱：get_money()
        转账：transfer_account()
        查看余额：check_money()
        查看明细：check_bill()
        修改密码：change_password()
        退出登录：login_out()
        注销用户：delete_user()
        运行：main()
- db 操作数据库，对数据的增删改查
    - db_handle.py 数据的增删改查
        存取数据：save_data(user_info)
        读取数据：select_data(phone_number)
        删除数据：delete_data(phone_number)
- lib 功能模块
    - common.py 常用功能
        登录判断：@is_login
        时间判断：nowtime()
- run.py 程序的运行文件
